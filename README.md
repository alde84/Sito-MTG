# AetherDeck 🃏

Piattaforma italiana per Magic: The Gathering — ricerca carte, creazione mazzi, storico prezzi Cardmarket e community.

---

## Struttura attuale

```
aetherdeck/
├── index.html              # Homepage
├── assets/
│   ├── css/
│   │   └── homepage.css    # Stili homepage
│   └── js/
│       └── homepage.js     # Script homepage
└── .gitignore
```

---

## Stack (prossimi step)

- **Web:** Next.js + React + TypeScript
- **Mobile:** React Native + Expo
- **Backend:** NestJS + Node.js
- **DB:** PostgreSQL + Redis
- **Monorepo:** Turborepo

---

## Come avviare il progetto (fase attuale)

Apri semplicemente `index.html` nel browser. Nessuna dipendenza richiesta per ora.
